## 💻 Getting started

### Requirements

**Clone the project and access the folder**

```bash
$ git clone https://github.com/joao-garcia404/pizzashop-web && cd pizzashop-web
```

**Follow the steps below**

```bash
# Install the dependencies
$ pnpm i

# Start the client
$ pnpm run dev
```
